# FACT: Focused Attention Consistency Training

**Mitigating Sycophancy and Jailbreaks via Attention-Level Invariance**

Stanford CS224N Custom Project — Emma Sampietro & Justin Hartenstein

FACT is a self-supervised post-training method that enforces attention-level invariance between clean and adversarially wrapped prompts. By regularizing attention patterns over shared core tokens, FACT reduces jailbreak ASR from 78% to 7% and sycophantic reasoning from 53% to 30% on Llama-3-8B-Instruct, while preserving general capabilities.

## Setup

Requires Python 3.11+, [uv](https://docs.astral.sh/uv/), a [Modal](https://modal.com/) account (A100-80GB), and an OpenAI API key (for StrongREJECT judge).

```bash
uv sync
echo "OPENAI_API_KEY=sk-..." > .env
modal setup
```

## Reproducing Results

All GPU experiments run on Modal. Commands below reproduce every table and figure in the paper.

```bash
# Prepare datasets (CPU, ~5 min)
modal run modal_app/run_experiment.py::prepare_data

# Attention patching experiment → Table 1
modal run --detach modal_app/run_experiment.py::patching
modal run --detach modal_app/run_experiment.py::patching --last-position-only

# Train FACT + CE-only baseline adapters → adapters saved to Modal volume
modal run --detach modal_app/run_experiment.py::fact_finetune
modal run --detach modal_app/run_experiment.py::ce_finetune

# Final evaluation (base vs CE vs FACT, 400 held-out test pairs + capability rollouts) → Figure 3
modal run --detach modal_app/run_experiment.py::final_eval

# Layer-skip ablation (trains 3 adapters: all layers / skip 6 / skip 12) → Table 2
modal run --detach modal_app/run_experiment.py::ablation_layer_eval

# Download results, then judge + compute CIs locally
modal volume get fact-data results/ ./results/
python experiments/judge_final_eval.py --dump results/final_eval_responses.jsonl
python experiments/compute_ablation_ci.py
```

### Local Smoke Tests (No GPU)

```bash
python experiments/run_baseline.py --n-samples 20 --device cpu
python experiments/run_fact.py --n-samples 20 --device cpu
```

## Structure

```
fact/               Core library (model hooks, data, training, patching, evaluation)
experiments/        Runnable scripts (baseline, patching, training, eval, judging)
modal_app/          Modal GPU entrypoints
data/scripts/       Data preparation
data/capability_evals/  Pre-computed IFEval + MMLU val sets
data/patching/      Pre-filtered informative pairs for patching experiment
```

## Key Hyperparameters

| Parameter | Value |
|-----------|-------|
| Learning rate | 1e-4 |
| FACT loss weight (lambda) | 0.25 |
| Epochs | 3 |
| Batch size | 2 |
| LoRA rank / alpha / dropout | 8 / 16 / 0.05 |
| LoRA targets | q, k, v, o, gate, up, down projections |
| FACT skip layers | 0-5 (first 6 of 32) |

## Model

[meta-llama/Meta-Llama-3-8B-Instruct](https://huggingface.co/meta-llama/Meta-Llama-3-8B-Instruct) (requires HuggingFace access approval).

## Data

See [data/README.md](data/README.md) for sources and formats. Datasets are generated by `prepare_data` from:
- **Jailbreaks**: [Mechanistic-Anomaly-Detection/llama3-jailbreaks](https://huggingface.co/datasets/Mechanistic-Anomaly-Detection/llama3-jailbreaks) (GCG, AutoDAN, multi-turn, prefill, many-shot)
- **Sycophancy**: [raybears/cot-transparency](https://github.com/raybears/cot-transparency) with strong assertive bias cues
- **Instruction following**: Cleaned Alpaca + IFEval format wrappers
